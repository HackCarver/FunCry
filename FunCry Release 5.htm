Overview
• FunCry is a randomizer mod for FarCry 2004. (requires patch 1.4)
• FunCry on ModDB
• FunCry on Discord
Bug reports? Coding genius and want to help? Contact FarOut or work on the issue directly in the open sourced build.
Installation
• Open your Far Cry folder. The location will vary, depending on what version you have
• Open the "Far Cry/Mods" folder. If needed, create the "Mods" folder
• Now copy the "FunCry" folder to the "Mods" folder..

^^ FunCry is now installed

• To start the FunCry mod, you can start FarCry ... and then select it from the mods list. The game will then restart and you'll be playing FunCry
• Alternatively, you can make a shortcut to FarCry.exe. Then you'd open the properties and append " -mod:FunCry" to the target properties


Effects
There's over 5000 random sounds that can be played. One is played when a new effect is started
Also there's a chance of getting a random skybox applied to each level. This feature can not be disabled
There's some basic effects for video cutscenes
Birds can be "hunted" now
Sound randomization capability can be disabled in general. (via config file)
Character and object randomization capability can be disabled in general. (via config file)
Conversation randomizer can be disabled via config file. These are randomized with other mercenary conversations
Radio messages are randomized when the "Conversation randomizer". They are randomized with other radio messages.
Languages (sounds from FCData\Localized) randomizer can be disabled via config file. This is a context free sound randomizer that is most active if the "Conversation randomizer" is disabled. Sounds from FCData\Localized are randomized with each other
Cutscene randomizer can be disabled via config file
Character randomizer can be disabled via config file
Carried/used weapon randomizer can be disabled via config file
Dropped weapon setting can be customized via config file

Config
You can change some things in the FunCry/Config.cfg file
• Time before another "rare" effect can be started
• Amount of effects active
• Amount of effects that are in the pending list
• Amount of pending effects that will be shown
• An alternative view
• Disable entire mod UI
• Disable individual effects [in case some are annoying, unwanted, or buggy]

And perhaps more. Just check the config file. Don't set the number of pending effects too high, or the game will lag a lot [and the log.txt file will say an effect could not be found]
Issues
• Certain combinations of effects can cause a visual glitch with the Hud. This does get reset eventually
• There may be a delay till you can fire a weapon after the fast motion effect
• Using the "Size matters" effect, you can scale certain items to a size that prevents you from progressing further. Such as the forklift in Factory level. Or doors on the Bunker level
Change Log
Jun 02 2021 Changes in fifth release
• "On the move..." effect now can restore the player to a previous position with a vehicle
• Turns out the "In the matrix" effect is uniquely bugged. Perhaps it only ever worked correctly for me. Oops
• Support for new categories of randomization. Characters can now be randomized, as well as weapons
• Weapons that the enemies carry and use can be randomized. This is done via config file
• Enemies can be forced to drop the weapon they would normally have... regardless of what they were using. This ability allows the player's weapons to be the normal ones while the enemies are doing some serious cheating. This is done via config file
• Cutscenes can be randomized. Kindof. However there is only one very small effect in this release
• New effect: "Is that you, Val?" .. This is an audio mod that changes mercenary voices to be that of Valerie
• This is the audio mod update. By far, the most time was spent on that. New internal structures, context sensitive replacement, etc. Many iterations of such things, as well as background projects that support this goal. Things like collecting basic attributes of each sound. That alone took quite some time and a custom Windows program. Long term, these investments in secondary work should enable richer and better quality audio mods.
• The config file has more options now, mainly releated to the new audio, character, and cutscene capabilites
• "In the matrix" effect is still enabled by default. It works for some, and looks terrible for others. Not enough feedback to know if it is generally ok, or generally buggy.
• "Can't run" effect has been renamed to "Out of breath"
• New effect: "Pulse"
• It has been a while since a FunCry build was released, so this one could have more issues than normal. Please report them if so
• Build 5 is the first public release of source code
